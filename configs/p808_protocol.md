# P.808 Subjective Test Protocol (Starter)

- Platform: AMT/Prolific with attention checks.
- Clips: 30–50, stratified by accent, condition, and codec/bitrate.
- Ratings: MOS (1–5) and CMOS (A vs B), with gold standard anchors.
- Exclusions: headphone check fail, RT < 0.3x media duration, failed honeypots.
- Deliverables: raw CSV of ratings (anonymized), analysis script, CIs per condition.
